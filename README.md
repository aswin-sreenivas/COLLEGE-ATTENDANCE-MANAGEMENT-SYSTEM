# 🎓 College Attendance Management System

A web-based application designed to simplify and automate the process of managing student attendance in colleges. The system enables administrators, faculty members, and students to access attendance-related information through a centralized platform.

---

## 📌 Overview

The College Attendance Management System (CAMS) helps educational institutions maintain attendance records digitally. It reduces manual work, improves accuracy, and provides quick access to attendance information.

---

## ✨ Features

- 🔐 Secure User Authentication
- 👨‍🎓 Student Management
- 👨‍🏫 Faculty Management
- ✅ Attendance Recording
- 📊 Attendance Report Generation
- 🔍 Search and Filter Records
- 📅 Attendance History Tracking
- 📱 Responsive User Interface

---

## 👥 User Roles

### Administrator
- Manage student records
- Manage faculty records
- Monitor attendance data
- Generate attendance reports

### Faculty
- Mark attendance
- View attendance records
- Manage assigned classes

### Student
- View attendance status
- Check attendance history

## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/College-Attendance-Management-System.git
```

### 2. Navigate to the Project Directory

```bash
cd College-Attendance-Management-System
```

### 3. Configure Database

- Create a MySQL database.
- Import the SQL file located in the `database` folder.
- Update database credentials in the configuration file.

### 4. Start the Server

Start Apache and MySQL using:

- XAMPP
- WAMP
- Laragon

### 5. Open in Browser

```text
http://localhost/College-Attendance-Management-System
```
